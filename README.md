# AdmissionPrep Social Assets

Public image host for AdmissionPrep social graphics (Instagram/TikTok carousels and posts)
produced by the `graphic-post-creator` skill. These are published marketing graphics — public by design.

Hosted here (GitHub raw, served via CDN) because Notion's image proxy reliably renders
`raw.githubusercontent.com` URLs, whereas catbox.moe links blank out inside Notion.

Each batch folder mirrors the skill's local `outputs/<date>-<slug>/` structure.
